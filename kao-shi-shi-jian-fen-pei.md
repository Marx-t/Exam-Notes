# 考试时间分配

## 考试安排

| 时间              | 科目     |
| --------------- | ------ |
| 第一天 9:00-11:30  | 语文     |
| 第一天 14:30-16:30 | 数学     |
| 第二天 9:00-11:00  | 物理化学合卷 |
| 第二天 14:30-16:30 | 英语     |
| 第三天 9:00-11:00  | 道法历史合卷 |

## 各学科时间分配

### 语文

| 题目       | 用时    |
| -------- | ----- |
| 实用类文本阅读  | 10min |
| 名著阅读     | 10min |
| 古诗、文言文阅读 | 15min |
| 语言综合运用   | 10min |
| 文学类文本阅读  | 35min |
| 作文       | 70min |

### 数学

30min内完成选填题，10、15可暂时先跳过

30min内完成17-21题

15-20min完成22题

15-20min完成23题，（3）可暂时先跳过

20-30min完成24题

预留5-15min检查

### 物理化学合卷

60-80min在试卷上完成所有题目（包括计算）

20-30min誊抄答题卡（检查并再算一遍）

10-15min检查（再算一遍）

### 英语

60-80min完成除作文外所有题目

30-40min完成作文

### 道法历史合卷

15-20min完成所有选择题

40min完成历史主观题

45min完成道法主观题

15min检查

### 值得注意的点

我没有在语文和英语考试中预留检查的时间，这并不表示检查在语文和英语中不重要，而是在语文和英语考试中极其容易出现一种情况——将对的答案改错！可见我血与泪的教训
